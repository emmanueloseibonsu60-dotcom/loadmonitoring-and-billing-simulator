# Electrical Load Monitoring and Billing Simulator

## Course Information
Programme: HND Electrical Engineering (Level 200)  
Course: EEE 227 (PT)  
Assessment Type: Project-Based Midterm Assignment  

## Project Description
This project simulates electrical load monitoring and electricity billing for a household or small facility. It allows users to register appliances, calculate energy consumption, and estimate electricity costs.

## Features
- Appliance registration and management
- Energy consumption calculation (kWh)
- Electricity billing calculation
- File storage using text files
- Menu-driven console interface
